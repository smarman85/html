Git Cheat Sheet

~:> git clone repo_from_git

Initialized the directory as a git repo

~:> git checkout -b branch-name

Makes a new branch off of the current branch

~:> git clone -d branch-name

Deletes the branch branch-name

~:> git branch -v

Shows available branches

~:> git stash

Stashes local changes so they aren't overwritten by a pull

~:> git stash apply

Applys the stashed changes after a pull

~:> git checkout -- track origin/branch-name

Makes a new branch that tracks origin

~:> git fetch upstream BRANCH

fetches changed from the upstream branch (usually master)

~:> git merge upstream/BRANCH

Merges Changes from upstream

~:> git remote add upstream ssh_form_upstream_git_repo

Addes the nickname "upstream" for the upstream repo. This is where changes will come from
